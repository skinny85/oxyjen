OxyjenZone

Free online service for hosting and sharing Oxyjen templates

Hosting

Each time you use the o2 command with a template like :name, the application will download the needed files from OxyjenZone.

Organizations

OxyjenZone stores each template within an organization. Creating your own is super easy - register through the web page or the command line by issuing ozone register.

Sharing

Once your Organization is registered and you are logged in, sharing a template is trivial - either using the Upload tab of the Organization panel, or through the command line by executing ozone push.

Discovering templates

Use the search functionality to discover new templates - both those from the official Oxyjen organization, and those uploaded by other users. If you shared your template, you should be able to find it there. You can also do it from the command line by issuing ozone search.
